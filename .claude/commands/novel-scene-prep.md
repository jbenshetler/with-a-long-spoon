Prepare a character and knowledge brief before writing a scene.

Gather what each character knows before a scene begins.

Steps:
1. Ask which chapter and which characters appear, if not already clear from context.
2. For each character: call `character_knowledge` for through_chapter=N-1 (what they know BEFORE this scene).
3. Call `research` for the scene location if relevant.
4. Query craft constraints for each character via `research` ("craft constraints for [character]").
5. Present a scene prep brief:
   - Each character's knowledge state entering the scene
   - Any craft constraints to keep in mind
   - Any unresolved temporal states relevant to this chapter
6. Do not write the scene. Present the brief and wait.
